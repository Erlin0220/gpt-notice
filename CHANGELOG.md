# Changelog

## 0.5.2 - 2026-07-23

- 修复项目主页切换到正式项目会话时，旧 DOM 会话 ID 覆盖新 URL 的问题。
- 页面切换时保留旧会话 URL，确保未完成队列由正确的后台标签接管。
- 输入框内容变化后立即刷新“加入队列”按钮状态。
- 复用已有会话标签时防止 Chrome 内存节省挂起；结束后恢复默认状态，不静音、不关闭用户标签。
- 增加项目会话路由、后台队列接管、完成清理和空闲禁入回归测试。

## 0.2.0 - 2026-07-23

- 修复最小化窗口无法判定任务完成。
- 任务存储从单一 `tabId` 升级为多标签绑定。
- 同一会话的运行中任务自动去重。
- 打开任务时复用现有会话标签。
- 避免页面初次加载历史消息时创建重复任务。
- 后台监控页增加周期刷新。
- 自动迁移 v0.1.0 的本地任务数据。

## 0.1.0 - 2026-07-23

- 初始版本。
